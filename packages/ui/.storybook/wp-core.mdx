# WordPress Core Components

The `@automattic/ui` package contains a number of still-private components from WordPress Core (`@wordpress/components`) that are forked for safe use in Automattic products. Simply import the components from the `@automattic/ui` package, and refer to the canonical [WordPress Components Storybook](https://wordpress.github.io/gutenberg) for more information.

Note that these forks are designed to be loaded safely alongside their original counterparts (e.g. no CSS class conflicts).

## `Badge` ([WP Storybook](https://wordpress.github.io/gutenberg/?path=/docs/components-badge--docs))

    ```jsx
    import { Badge } from '@automattic/ui';
    ```

## `Menu` ([WP Storybook](https://wordpress.github.io/gutenberg/?path=/docs/components-menu--docs))

    ```jsx
    import { Menu } from '@automattic/ui';
    ```

## `Tabs` ([WP Storybook](https://wordpress.github.io/gutenberg/?path=/docs/components-tabs--docs))

    ```jsx
    import { Tabs } from '@automattic/ui';
    ```
